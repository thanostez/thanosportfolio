# ⚡ Thanos Portfolio — Modern Cyberpunk Edition

A high-performance, single-page developer portfolio designed with a futuristic **Cyberpunk / Hacker** aesthetic. Built for speed, reliability, and visual impact.

![Thanos Portfolio Preview](https://s0.wp.com/mshots/v1/https%3A%2F%2Fthanoslab.site?w=1280&h=720)

## 🚀 Key Features

- **Cyberpunk Aesthetics**: Dark-themed UI with neon accents (Cyan, Pink, Green, Purple) and glassmorphism components.
- **Mobile-First Experience**: Fully responsive layout with a modern, glassy **Bottom Navigation Bar** for mobile users.
- **9× Live Project Showcases**: Featuring legal-tech, 10+ Telegram bots, Web3 platforms, and security tools with real-time screenshot previews.
- **Dedicated Apps Section**: Integrated APK download section for mobile utilities like DripJudge AI, Toolzy, and Fokus.
- **Dynamic Skill Tracking**: Animated skill bars and tech-stack categorization.
- **Integrated Contact System**: Production-ready contact form powered by **Formspree** with AJAX submission and live feedback.
- **Performance Optimized**: Vanilla tech stack for near-instant load times, minimal dependencies, and SEO best practices.
- **Custom Branding**: Fully branded with a custom-generated AI favicon and professional meta-data.

## 🛠️ Tech Stack

- **Core**: HTML5, Semantic Markup
- **Styling**: Vanilla CSS3 (Grid, Flexbox, Advanced Animations, Glassmorphism)
- **Logic**: Vanilla JavaScript (ScrollSpy, Intersection Observer API, dynamic particles)
- **APIs**: WordPress mshots (Dynamic screenshots), Formspree (Form handling)
- **Typography**: Google Fonts (Orbitron, Space Mono)

## 📦 Getting Started

### Local Development
The portfolio is built using a portable vanilla stack. No build step is required.

1. Clone the repository:
   ```bash
   git clone https://github.com/thanostez/thanosportfolio.git
   ```
2. Open `index.html` in any modern web browser..

### Configuring the Contact Form
To receive messages in your inbox:
1. Sign up at [Formspree.io](https://formspree.io).
2. Create a "New Form" and copy your unique endpoint ID.
3. Replace the `ENDPOINT` constant in the `handleSubmit` function within `index.html`:
   ```javascript
   const ENDPOINT = 'https://formspree.io/f/YOUR_ID_HERE';
   ```
